Jrotate

A simple jQuery plugin that animate rotations

Download this project as a .zip file Download this project as a tar.gz file

What is it?

In jQuery, you can animate a moving element, from left to right, for example, but it is impossible to animate rotation.
Now, with jRotate, your rotation is no longer instantaneous, but degree by degree. Just set the angle you want to add (or remove), the duration of the animation and it's on !
